### ABOUT

this is a project is based on [howtographql](https://www.howtographql.com/) to study how to work with react and graphql with apollo on the frontend

### HOW TO RUN

_with npm_:

```sh
$ npm i

$ npm start
```

_with yarn_:

```sh
$ yarn

$ yarn start
```
